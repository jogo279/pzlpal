Crossword Puzzle Solver

PzlPal is a crossword puzzle solver which turns pictures into solved puzzles in under a minute. There are three steps: first you upload a picture of a puzzle (for example, from your newspaper or a website). Next, you help us break the image into components, namely the grid and the clues. Finally, we present the digitized puzzle, where you can make final adjustments and then let us solve your puzzle!

Origins

PzlPal was originally created for Carnegie Mellon University's yearly hackathon, TartanHacks. PzlPal won first place in TartanHacks 2014: you can read the write up in Carnegie Mellon Today. This website is a more fleshed-out version of the original hack.

Technology

PzlPal is written primarily in NodeJS using the MEAN stack. Image processing is performed using OpenCV, and optical character recognition via NewOCR's API and Tesseract. To solve the puzzles, we first obtain possible answers for each slot by running a full-text search on Matt Ginzer's Cluer Database using Apache Solr. We use these answers to fill the grid using a backtracking algorithm, modified somewhat for efficiency. PzlPal is hosted on DigitalOcean.
