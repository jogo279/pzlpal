Origins

PzlPal was originally created for Carnegie Mellon University's yearly hackathon, TartanHacks. PzlPal won first place in TartanHacks 2014: you can read the write up in Carnegie Mellon Today. This website is a more fleshed-out version of the original hack.

Technology

PzlPal is written primarily in NodeJS using the MEAN stack. Image processing is performed using OpenCV, and optical character recognition via NewOCR's API and Tesseract. To solve the puzzles, we obtain possible answers for each slot by running a fuzzy full-text search on Matt Ginzer's Cluer Database. PzlPal is hosted on DigitalOcean.
